What is QV exactly? It's a text editor built upon the pretty popular Kilo Editor, but with a lot more added. Here's a short list on everything added:
* More syntax highlighting
* Config file
* Line numbers
* Better colourscheme
It's pretty good, actually! You can install it with:
> git clone https://github.com/ElisStaaf/qv ~/qv
Then, you wanna use make to build to system:
> cd ~/qv
> make
Then, you're free to use your editor with:
> qv $file $config
Enjoy!
